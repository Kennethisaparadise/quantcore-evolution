## Polymarket API Research
- Gamma API (public): https://gamma-api.polymarket.com
  - /events - List active events
  - /markets - Market details
  - /sports - Sports leagues
  - /tags - Categories
- CLOB API: https://clob.polymarket.com
  - /price - Get token price
  - /book - Order book depth
- No API key required for market data (public endpoints)
- Wallet/private key only needed for actual trading
